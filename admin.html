📊 Google Sheets 연동

상담신청, 제휴문의 폼 데이터가 Google Sheets에 자동 저장됩니다.

📋 설정 방법

  1. Google Sheets에서 새 스프레드시트 생성
  2. 확장 프로그램 → Apps Script 클릭
  3. 아래 코드를 붙여넣고 저장
  4. 배포 → 새 배포 → 웹 앱 선택
  5. 액세스 권한: "모든 사용자"로 설정
  6. 생성된 URL을 위 필드에 입력 후 저장
